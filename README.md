## 🧾 Introduction

Welcome to the **Family Expense Tracker** 👋  
This is a simple Python project that helps track family income and expenses.

It allows you to:
- Add family members  
- Record their earnings  
- Track expenses  
- Calculate remaining balance  

---

## ✨ Features

- Add family members easily  
- Record income for each member  
- Track daily/monthly expenses  
- Automatic balance calculation  
- Simple and clean interface  

---

## 🛠️ How to Use

### 1️⃣ Add Family Members
- Run the application  
- Enter member names  

### 2️⃣ Record Earnings
- Add income for each member  

### 3️⃣ Track Expenses
- Add expenses like food, bills, etc.  
- Balance updates automatically
